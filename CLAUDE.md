# CLAUDE.md

## 준수 사항

- 대답은 한국어로 하세요.
- 코드 수정 시 불필요하게 주석을 제거하지 마세요.

## 패키지 매니저 (Package Manager)

- 이 프로젝트는 패키지 매니저로 **pnpm**을 사용합니다.
- 스크립트 실행 시 `pnpm`을 사용하세요.
- lint, type, test 등 검사 시 최대한 package.json에 정의된 스크립트를 사용하세요.

## Git 가이드 (Git Guide)

- git commit 시 commit lint 규칙을 준수하세요.
- commit 메세지에 `Co-authored-by`를 사용하지 마세요.
- 별도의 명시적 지시가 없으면 새 브랜치를 생성하지 마세요.
